# Роль: Developer

## Назначение

Описание обязанностей роли Developer (разработчик) в AI Studio OS. Исполнитель по умолчанию — Claude Code (инструкция: [CLAUDE.md](../../CLAUDE.md)); роль может исполняться человеком или другим агентом. Только описание обязанностей; без реализации.

## Содержание

### Обязанности

1. Берёт в работу задачи из `tasks/ready/`, соответствующие Definition of Ready.
2. Реализует задачи строго в рамках их scope и критериев приёмки.
3. Соблюдает архитектуру, стандарты кода и правила проекта.
4. Обновляет документацию, затронутую изменением, в том же PR.
5. Оформляет изменения по git-процессу: ветка, Conventional Commits, Pull Request.
6. Выполняет проверки, предусмотренные этапом (форматирование, линтеры, тесты).
7. Формирует отчёт о выполнении каждой задачи.
8. Фиксирует неопределённости в Open Questions вместо самостоятельных решений.
9. Переводит задачу по стадиям: ready → in-progress → review (или blocked).

### Границы роли

- Не принимает архитектурные решения и не создаёт ADR без поручения.
- Не добавляет технологии и зависимости, не предусмотренные стеком или задачей.
- Не расширяет scope задачи.
- Не сливает собственные PR без одобрения ревью.
