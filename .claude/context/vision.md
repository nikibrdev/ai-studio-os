# Контекст: Видение

## Назначение

Краткое видение продукта для быстрой загрузки контекста AI-агентом перед работой. Источник истины — [README.md](../../README.md) и [CONSTITUTION.md](../../CONSTITUTION.md).

## Содержание

### Что мы строим

**AI Studio OS** — open-source платформа управления разработкой ПО, в которой роли команды (Project Manager, Developer, QA Engineer) исполняются людьми **или AI-агентами** по единому процессу.

### Ключевые тезисы

1. Роль отделена от исполнителя: процесс одинаков для человека и агента.
2. Ядро независимо от AI-провайдера; Claude Code — Developer по умолчанию, другие агенты подключаются через адаптеры.
3. Процесс прозрачен: задачи, события, отчёты и решения фиксируются и наблюдаемы.
4. Качество выше скорости; документация первична.

### Что продуктом НЕ является

- Не «автопилот», бесконтрольно пишущий код: человек всегда видит и контролирует процесс.
- Не обёртка над одной AI-моделью.
- Не замена git/GitHub — платформа встраивается в стандартный процесс разработки.

### Текущий этап

v0.1 Foundation: только структура и документация. Roadmap: [ROADMAP.md](../../ROADMAP.md).
