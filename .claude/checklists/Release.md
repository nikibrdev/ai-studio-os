# Чек-лист: Release

## Назначение

Проверка готовности версии к релизу. Используется мейнтейнерами начиная с первых версий с работающей функциональностью; пункты, зависящие от CI/CD и эксплуатации, применимы после их появления.

## Содержание

### Готовность содержания

- [ ] Все задачи версии в `tasks/done/`; незавершённые перенесены осознанно.
- [ ] Цели версии из [ROADMAP.md](../../ROADMAP.md) достигнуты либо отклонения задокументированы.
- [ ] Все ADR версии в статусе «Принято»; Open Questions версии закрыты или перенесены.

### Качество

- [ ] Все тесты проходят (unit, интеграционные, e2e — по мере появления уровней).
- [ ] Известные дефекты триажированы: блокирующих нет, остальные зафиксированы задачами.

### Документация

- [ ] CHANGELOG.md: раздел Unreleased переведён в раздел версии с датой.
- [ ] README и архитектурная документация соответствуют состоянию версии.
- [ ] Документация эксплуатации обновлена (после появления `docs/operations/`).

### Публикация

- [ ] Версия помечена git-тегом по Semantic Versioning.
- [ ] Релиз оформлен на GitHub с описанием изменений.
