# Модуль: internal/domain/shared

## Назначение

Язык предметной области — словарные типы, общие для всех доменных модулей: Role, TaskState; по мере принятия решений — идентификаторы (ADR-011), доменные ошибки, value objects ([ADR-015](../../../docs/adr/ADR-015-internal-layering.md)).

## Содержание

### Ответственность

- Role — 5 ролей MVP ([domain-model.md](../../../docs/architecture/domain-model.md)).
- TaskState — 9 канонических состояний ([state-machine.md](../../../docs/architecture/state-machine.md)).
- Только типы и константы; никакой логики.

### Зависимости

- Разрешено: только стандартная библиотека.
- Запрещено: всё остальное, включая `internal/platform` — словарь домена ни от чего не зависит.

### События

Не публикует и не потребляет; имена типов событий — [domain/event](../event/README.md).

### Ограничения

Изменение словаря (новое состояние, новая роль) — изменение замороженной архитектуры: только через ADR.
