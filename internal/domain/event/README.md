# Модуль: internal/domain/event

## Назначение

Словарь типов событий платформы: константы имён всех событий, один к одному с каталогом [events.md](../../../docs/architecture/events.md).

## Содержание

### Ответственность

- Константы 15 типов событий (10 основных + 5 дополнительных); подписки используют константы, а не строковые литералы.
- Схемы payload сюда не входят — они принадлежат модулям-источникам (Domain Layer).

### Зависимости

- Разрешено: stdlib (фактически — ничего).
- Запрещено: всё остальное.

### События

Определяет имена; не публикует и не потребляет. Контракт шины — `internal/core` (EventBus); журнал и доставка — Infrastructure ([ADR-002](../../../docs/adr/ADR-002-event-delivery.md)).

### Ограничения

Имя события неизменяемо после появления в каталоге; изменение структуры payload — новая версия схемы (Immutable Events, [engineering-principles.md](../../../docs/architecture/engineering-principles.md)).
