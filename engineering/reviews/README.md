# Engineering: Reviews

## Назначение

Записи code review: какие замечания были высказаны, какие решения приняты, как исправлено. Источник обучения для агентов и людей: повторяющиеся замечания — сигнал для правила в стандартах.

## Содержание

Формат записи: контекст (что ревьюилось), замечания (нумерованные), решение по каждому, статус исправления. Имя файла: `YYYY-MM-DD-<subject>.md`.
