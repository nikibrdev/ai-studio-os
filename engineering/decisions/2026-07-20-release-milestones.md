# Решение: релизные вехи (Roadmap v2)

## Назначение

Фиксирует пересмотр [ROADMAP.md](../../ROADMAP.md) архитектором проекта (2026-07-20): версии переопределены по слоям архитектуры, а не по ролям, как в первоначальном черновике этапа Foundation.

## Содержание

### Контекст

Первоначальный roadmap (2026-07-19) был написан до проектирования архитектуры и описывал версии как последовательность ролей (Project Management → Developer Engine → QA Engine → Workflow → ...). После Architecture Freeze стала известна реальная структура: `domain → application → platform → infrastructure` ([ADR-015](../../docs/adr/ADR-015-internal-layering.md)). Управление прогрессом только по задачам недостаточно — нужна оценка готовности продукта.
