# agents/ — адаптеры исполнителей

## Назначение

Реализации `platform.Executor` ([ADR-005](../docs/adr/ADR-005-executor-contract.md)) — конкретные технические бэкенды (Claude Code, в будущем — другие), которые платформа запускает для исполнения ролей. Один каталог — один адаптер; адаптеры не знают друг о друге ([module-boundaries.md](../docs/architecture/module-boundaries.md)).

## Содержание

### Ограничения (по `module-boundaries.md`)

- **Разрешено:** опубликованный контракт `platform.Executor`; `pkg/`; SDK/CLI своего провайдера.
- **Запрещено:** импорт внутренностей `internal/` (включая `internal/infrastructure` — Docker CLI внутри адаптера вызывается напрямую через `os/exec`, не через инфраструктурный слой платформы); прямой доступ к хранилищам платформы; знание о других адаптерах.
