# TASK-008: Типы событий (internal/events)

## Тип

feature

## Эпик

EPIC-002 ([docs/roadmap/EPIC-002-foundation.md](../../docs/roadmap/EPIC-002-foundation.md))

## Цель

Пакет `internal/events` с идентификаторами типов всех 15 событий каталога [events.md](../../docs/architecture/events.md) — константы, один к одному с документацией.

## Контекст

Схемы (payload) событий — Domain Layer; здесь фиксируется только словарь типов, чтобы подписки и каталог опирались на константы, а не на строковые литералы.

## Scope

### Входит

- `internal/events/types.go`: константы 15 типов событий (10 основных + 5 дополнительных) с doc-комментариями и ссылкой на каталог.

### Не входит

- Структуры payload; реализация шины; журнал.

- [ ] Только константы.
