# Стадия: Ready

## Назначение

Каталог задач, готовых к взятию в работу: Definition of Ready выполнен полностью.

## Содержание

### Критерий попадания

Задача полностью оформлена по шаблону [Task.md](../../.claude/templates/Task.md): цель, критерии приёмки, scope, затрагиваемые модули, ограничения. DoR-чеклист внутри задачи закрыт. Проверяет и переводит — Project Manager.

### Правила

1. Исполнитель роли Developer берёт задачи **только** из этого каталога.
2. Задача из `ready/` не изменяется по существу; если требования изменились — задача возвращается в `backlog/` на доработку.

### Переходы

- **→ `in-progress/`** — Developer взял задачу в работу.
- **→ `backlog/`** — требования изменились, DoR больше не выполнен.
