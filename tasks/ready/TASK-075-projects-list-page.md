# TASK-075: Страница списка проектов

## Тип

feature

## Эпик

[EPIC-009 Dashboard](../../docs/roadmap/EPIC-009-dashboard.md)

## Цель

Первая страница с настоящими данными: список проектов через `GET /projects`.

## Контекст

Поверх каркаса TASK-074 (API-клиент, layout) и `GET /projects` (TASK-072/073).

## Scope

### Входит

- `/` — список проектов: имя, состояние (`created`/`active`/`archived`), ссылка на страницу проекта (TASK-076).
- Состояния загрузки/ошибки/пустого списка.
- Компонентный/страничный тест (Vitest + React Testing Library) на моке HTTP-ответа `GET /projects`.

### Не входит

- Страница проекта — TASK-076.

## Критерии приёмки

- [ ] Страница отображает реальные данные при запуске против настоящего `apps/api`.
- [ ] Пустой список проектов не считается ошибкой (отдельное сообщение, не падение страницы).
- [ ] `pnpm test` — тест страницы зелёный.

## Затрагиваемые модули и документы
