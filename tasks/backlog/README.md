# Стадия: Backlog

## Назначение

Каталог идей и задач, которые зафиксированы, но ещё не подготовлены к работе.

## Содержание

### Что здесь лежит

Задачи в любой степени готовности: от одной строки идеи до почти готовой задачи, не прошедшей Definition of Ready.

### Правила

1. Добавить задачу в backlog может любая роль; управляет backlog — Project Manager.
2. Задача оформляется по шаблону [Task.md](../../.claude/templates/Task.md) (допускается частичное заполнение).
3. Порядок приоритетов определяет Project Manager.

### Переходы

- **→ `ready/`** — когда Project Manager довёл задачу до Definition of Ready.
- **→ `archive/`** — если задача потеряла актуальность.
