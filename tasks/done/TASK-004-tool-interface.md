# TASK-004: Интерфейс Tool

## Тип

feature

## Эпик

## Цель

Контракт инструмента в `internal/core`: ToolDescriptor, Tool — по [interfaces.md](../../docs/architecture/interfaces.md) и [tools.md](../../docs/architecture/tools.md).

## Контекст

Слой инструментов детально проектируется в v0.8; сейчас фиксируется минимальный контракт вызова. По [ADR-014](../../docs/adr/ADR-014-module-interaction.md) инструмент не вызывает Core — контракт не содержит зависимостей в сторону модулей ядра.

## Scope

### Входит

- `internal/core/tool.go`: интерфейсы ToolDescriptor, Tool; doc-комментарии (одно действие, явная регистрация, журналирование вызовов — ограничения контракта).

### Не входит

- Формат деклараций и схема параметров (v0.8); реестр инструментов (модуль `tool`, Domain Layer); реализации.

- [ ] Только интерфейсы; запрет Tool → Core отражён в doc-комментарии.
