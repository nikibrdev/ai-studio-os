# Стадия: Done

## Назначение

Каталог выполненных задач: Definition of Done подтверждён, изменения слиты в основную ветку.

## Содержание

### Критерий попадания

Definition of Done ([CONSTITUTION.md](../../CONSTITUTION.md)) выполнен полностью: критерии приёмки закрыты, ревью пройдено, PR слит, документация обновлена, отчёт о выполнении есть в файле задачи. Переводит задачу — QA Engineer или ревьюер.

### Правила

1. Файлы задач в done не редактируются (кроме пометки об архивировании).
2. Done-задачи — источник истории проекта: отчёты и решения из них используются при анализе и планировании.

### Переходы

- **→ `archive/`** — по решению Project Manager (например, после завершения версии).
