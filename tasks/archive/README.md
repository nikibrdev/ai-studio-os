# Стадия: Archive

## Назначение

Архив задач: завершённые задачи прошлых версий и задачи, потерявшие актуальность.

## Содержание

### Что здесь лежит

- Done-задачи, перенесённые после завершения версии.
- Отменённые задачи из `backlog/` и `blocked/` — с записью причины отмены в истории задачи.

### Правила

1. Архивирование выполняет Project Manager.
2. Файлы в архиве не редактируются и не удаляются: архив — часть истории проекта.
3. Возврат из архива — создание новой задачи со ссылкой на архивную, а не «реанимация» файла.
